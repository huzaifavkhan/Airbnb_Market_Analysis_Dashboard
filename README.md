# **Airbnb Market Analysis Dashboard** 

This Power BI project explores Airbnb open data from New York City to uncover insights about the short-term rental market. Using data from Kaggle, it provides an interactive dashboard with visuals highlighting:

* **Total Activity**: Summarized metrics like total reviews (1.1 million+), total listings (48,895), total hosts (37,457), average nightly prices (\$152), and average minimum nights (7).
* **Neighborhood Analysis**: Top neighborhoods by reviews, price distribution by area, and varying minimum stay requirements across room types and boroughs.
* **Property Insights**: Listings with the highest reviews, time-series trends in average prices and property counts, and room type breakdowns.
* **Host Profiles**: Data on host engagement, host growth over time, and host pricing strategies, including top hosts by review count and pricing.
* **Design Thinking Framework**: The report applies a user-centered approach to understand challenges faced by hosts and guests, integrating feedback for dashboard improvements.
